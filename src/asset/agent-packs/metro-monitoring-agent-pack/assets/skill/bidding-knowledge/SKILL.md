## 适用场景

- 编制工程监测投标技术标和商务标
- 采集招标公告/招标文件情报，判断是否值得跟进
- 分析招标文件评分细则，制定投标策略
- 审核投标文件完整性和合规性
- 项目报价测算与成本分析

## 投标文件标准结构

### 技术标核心章节

| 序号 | 章节 | 评分权重（典型） | 编制要点 |
|-----|------|----------------|---------|
| 1 | 项目理解与分析 | 10~15分 | 体现对项目特点、重难点的深入理解，切忌泛泛而谈 |
| 2 | 监测方案 | 25~35分 | 测点布设图、仪器选型、监测频率、数据处理方案 |
| 3 | 质量保证措施 | 10~15分 | ISO体系、三级校核制度、数据复核流程 |
| 4 | 安全文明施工 | 5~10分 | 安全教育、应急预案、文明施工措施 |
| 5 | 工期保证 | 5~10分 | 人员及设备投入计划、关键节点时间表 |
| 6 | 项目业绩 | 10~15分 | 类似工程合同+完工证明，优先选择地铁/基坑类 |
| 7 | 人员配置 | 10~15分 | 项目负责人资质（注册测绘师）、技术骨干经验 |
| 8 | 应急预案 | 5~10分 | 超限应急响应流程、24小时值班制度 |

### 商务标核心要素

| 要素 | 说明 |
|-----|------|
| 报价构成 | 人工费 + 仪器折旧/租赁费 + 交通差旅费 + 管理费 + 利润 + 税金 |
| 报价策略 | 基准价浮动法：以成本估算为基础，根据竞争态势调整下浮率 |
| 不平衡报价 | 前期投入项（基准点埋设、方案编制）可适当提高单价，后期常规监测按次计费 |
| 资质要求 | 工程测量甲级/乙级、测绘资质、ISO认证、安全生产许可证 |

## 投标常见扣分项（避坑清单）

1. **未响应招标文件的特殊要求**（如指定精度等级、特定仪器品牌）
2. **监测方案缺少测点布设图**（评审专家最关注的直观材料）
3. **人员资质证书过期或与投标项目无关**
4. **业绩证明缺少合同关键页**（合同首页+签字页+金额页+完工证明）
5. **报价明显低于成本价**（可能被判定为低于成本价投标而废标）
6. **未单独编制应急预案章节**（部分招标文件明确要求独立章节）
7. **投标文件页码、目录与正文不对应**

## 评标专家关注重点

1. **方案针对性**：是否针对本项目特点定制，而非模板套用
2. **测点布设合理性**：点位密度是否满足规范，关键部位是否加密
3. **仪器选型匹配度**：精度等级是否满足监测要求
4. **团队实力**：项目负责人是否有同类项目经验
5. **应急能力**：是否有完善的应急预案和快速响应机制
6. **报价合理性**：不追求最低价，而是性价比和工作量覆盖度

## 投标情报预审表

收到招标公告、链接或招标文件摘录时，先输出下表，再进入正式投标文件编制：

| 字段 | 判定方法 |
|---|---|
| 项目匹配度 | 对比公司地铁保护区、基坑、控制测量、自动化监测业绩 |
| 资质差距 | 核查测绘/勘察资质、人员证书、ISO、安全生产等要求 |
| 业绩匹配 | 标注可用业绩、缺失业绩和证明材料缺口 |
| 技术重难点 | 判断保护区距离、施工工况、自动化投入、专家评审风险 |
| 商务风险 | 付款节点、保证金、履约保证、低价风险、工期压缩 |
| 建议动作 | 跟进、谨慎跟进、补资料后判断、放弃 |

无法从资料中核验的信息必须写“待核验”，不得补写公告没有给出的预算、评分办法或截止时间。

## 结算对账核验

催款前先做对账：

| 核验项 | 说明 |
|---|---|
| 合同金额 | 合同总价、暂定价、单价合同或补充协议 |
| 已确认产值 | 监测期数、报告数量、工作量签认单 |
| 已开票金额 | 发票号、开票日期、税率 |
| 已收款金额 | 银行流水或甲方付款记录 |
| 应收余额 | 按付款节点计算，不只按合同总价减已收 |
| 差额原因 | 未开票、已开票未付、付款条件未触发、扣款、口径不一致 |

金额不一致时优先输出《结算对账说明》或《对账确认函》，不要直接生成强硬催款函。
