# 内审 / 专家评审意见回复表格式

宁波市轨道交通集团及大部分轨道公司在方案返修时，**必须随方案附带一份"意见回复表"**。本文件给出标准格式与高质量回复样例。

## 一、内审回复表模板

```markdown
关于《{方案全名}》内审修改建议的回复

工程名称：{工程全名}

内审修改建议及回复：

1、{原意见 1 原文照抄}
回复：{修改后的具体描述}，详见 P{页码}（或第 X.Y 章节）。

2、{原意见 2 原文照抄}
回复：{修改后的具体描述}，详见 P{页码}。

3、{原意见 3 原文照抄}
回复：{修改后的具体描述}，详见 P{页码}。

修改确认：（签字）
审核确认：（签字）

{编制单位名称}
{年}    {月}    {日}
```

## 二、专家评审回复表模板（表格形式）

```markdown
{工程全名}监测方案专家评审意见回复

| 序号 | 专家意见 | 意见回复 |
|---|---|---|
| 1 | {原意见 1} | {回复内容}，详见 P{页码} |
| 2 | {原意见 2} | {回复内容}，详见 P{页码} |
| 3 | {原意见 3} | {回复内容}，详见 P{页码} |

修改确认签字：__________
{年}    {月}    {日}
```

## 四、回复书写要点（必看）

### 1. 三必有
- **"已"字开头**：回复必须以"已 ... " 开头表明已修改（不要写"将"或"会"）
- **明确页码**：每条回复**必须注明页码**（详见 PXX）或章节（第 X.Y 节）—— 评审硬性要求
- **对应具体动作**：不能笼统说"已修改"，必须说**修改了什么内容**

### 2. 三避免
- 避免承诺式语言（"将考虑"、"后续优化"）
- 避免**未实际修改就回复**——评审会逐条翻页核对
- 避免**多条意见合并回复**——必须一条一回复

### 3. 高分回复模板
```
回复：已按会议要求{具体修改动作}：
（1）在 X.Y 节{修改点 1}；
（2）增加图 X.Y-N {新增图名}；
（3）{其他修改}；
详见 P{起始页}–P{结束页}。
```

## 五、回复表附在方案的位置

- **方案最前面**：紧跟在封面、签章页之后，目录之前
- 一般顺序：封面 → 签章页（编制/校核/审核/批准）→ 内审意见 → **内审意见回复** → 专家评审意见 → **专家评审意见回复** → 目录 → 正文
