# 模板 A：运营分公司标准格式（16 章，重型）

适用于已运营线路（如宁波 1/2/3/6/7/8 号线运营段）受外部施工影响时的监测方案。**章节编号与顺序为强制约束**，运营分公司技术委员会严格按此目录评审。

## 章节级别注意点

- **章节 1.5 施工筹划** 必须有日期化的横道图或工筹表
- **章节 1.7 区间沉降数据** 引用既有运营监测的工后变形（mm/年），单位与日期必须明确
- **章节 1.8 安全评估** 必须明确：风险等级、最大变形预测值、关键工序、加固方案
- **章节 3.4 监测频率** **必须分阶段独立成表**，每个施工阶段一张表
- **章节 11.1.1 报警值** **必须细化到各施工阶段**，并区分 预警值/报警值/控制值 三栏
- **章节 16 附图** 是评审最关注的内容，必须含：总平图、位置关系图、各阶段布点图、断面详图、控制网示意图
