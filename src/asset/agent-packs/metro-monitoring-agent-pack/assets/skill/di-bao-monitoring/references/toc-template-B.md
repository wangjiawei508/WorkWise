# 模板 B：设计院精简格式（11–14 章，轻型）

适用于建设期/未运营线路、由设计院出具方案的轻量级版本。代表：甬江科创启动区 3# 地块方案、通途西路综合管廊方案。

## 完整目录

```
1 项目概况
    1.1 工程概述
    1.2 设计概况（围护、支护、止水、施工顺序等）
    1.3 位置关系                           ← 与轨道交通结构的关系
    1.4 地质及周边环境
    1.5 施工筹划及顺序

2 安全评估及现状普查
    2.1 风险等级与模拟评估
    2.2 隧道结构拟产生变形评估             ← 引用安评报告
    2.3 工后变形监测数据
    2.4 隧道现状普查

3 监测实施
    3.1 监测目的及规范依据
    3.2 监测人员及仪器
    3.3 监测等级
    3.4 监测重难点及应对措施
    3.5 监测实施
        3.5.1 监测范围
        3.5.2 监测项目及测点数量
        3.5.3 监测周期
        3.5.4 监测频率                     ← 通常仅 1–2 张表
    3.6 测点布设
        3.6.1 平面控制点布设
        3.6.2 高程控制点布设
        3.6.3 监测点布设
    3.7 监测报警值与停测标准
        3.7.1 监测报警值
        3.7.2 停测标准
    3.8 监测作业计划进度
    3.9 预消警流程

4 控制网建立与复测
    4.1 坐标系统及高程基准
    4.2 平面控制网
    4.3 高程控制网
    4.4 自动化监测控制网

5 监测方法与数据处理
    5.1 控制网测量
    5.2 自动化监测
    5.3 人工监测
    5.4 保护区基坑抽测（如有）

6 数据处理及变形分析
    6.1 数据处理
    6.2 变形分析

7 巡视巡查

8 应急预案及作业管理
    8.1 应急响应流程
    8.2 应急响应组织机构及职责
    8.3 应急设备
    8.4 防汛、防台专项应急预案
    8.5 轨行区作业管理（运营线路才有）

9 监测信息反馈
    9.1 信息反馈的方式
    9.2 信息反馈时间
    9.3 资料归档

10 与工程相关单位的配合措施
    10.1 与业主单位的配合服务
    10.2 与施工现场的配合服务

11 其他说明
    11.1 数据保密
    11.2 其他说明

12 附图
```

## 与模板 A 的差异说明

| 项目 | 模板 A（运营版） | 模板 B（精简版） |
|---|---|---|
| 章节总数 | 16 | 11–14 |
| 编制依据 | 单独列 2.2 | 合并到 3.1 监测目的及规范依据 |
| 监测重难点 | 单独列 2.3 | 列在 3.4 |
| 控制网 | 单独列第 4 章 | 仍单独 1 章但合并 5 小节 |
| 监测点布设 | 单独列第 5 章 | 合并到 3.6 |
| 报警值 | 单独列第 11 章+预消警流程 | 列在 3.7+3.9 |
| 轨行区作业管理 | 单独列第 10 章 | 列在 8.5（仅运营线路才有） |
| 保障措施 | 单独列第 13 章 | 弱化或不列 |
| 与各方配合 | 单独列第 14 章 | 列第 10 章 |

## 使用建议

- 用户若提到**建设期线路**或**第三方监测/小规模**，默认用此模板
- 若运营单位明确要求按"地保监测监护标准化手册"格式，**仍需切换到模板 A**
- 即便用 B 模板，章节 2 安全评估、3.4 监测重难点、3.7 报警值仍是评审重点，不能简化
