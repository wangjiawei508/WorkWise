# 内审 / 专家评审高频意见自检清单

本清单汇总了 9 份方案在**内审会议纪要、专家评审意见、修改回复表**中**反复出现的修改意见**。每条意见对应方案中应当满足的"要件"。**交付前请逐条打勾，缺哪项就补哪项**。

## 一、工程概况与位置关系（第 1 章）

- [ ] 工程总平面图**清晰显示新老结构施工范围**及与既有轨道交通车站、区间、附属、顶管通道的**位置关系**（不能只放一张大概的红线图）
- [ ] 周边环境节中**补充重载车辆停放/行走路线位置**
- [ ] 1.3 工程及水文地质有勘察报告关键参数表
- [ ] 1.5 施工筹划有**日期化横道图或工筹表**，按工序细化分部工程
- [ ] 1.7 区间沉降数据：**补充车站部分工后沉降监测数据**
- [ ] 1.8 安全评估引用**安评报告**结论：风险等级、最大变形预测值
- [ ] 附属结构、顶管通道、风井等**单独列出位置**（不能只在文字描述）
- [ ] 既有线**加固情况**（如盾构加固、桩位）有专门小节说明
- [ ] 6 号线方案要求："补充 X 号线 X 站施工总平布置图"——所有交叉换乘站都要总平图

## 二、监测目的、编制依据、重难点（第 2 章）

- [ ] 编制依据规范**不少于 16 部**（国家 8 + 地方行业 5 + 项目专属 3+）
- [ ] 2.3 监测重难点**针对本项目实际**（不可与其他项目雷同），并有对应"应对措施"
- [ ] 重难点应"**突出本项目的重中之重**"（评审常见意见）

## 三、监测范围、项目、频率（第 3 章）

- [ ] 监测范围明确**正投影范围 + 两侧延伸值**（基坑：3H；顶管：30 环）
- [ ] **重点监测区**明确并单独标识（如顶管 D 范围 5 环加密 1 个断面）
- [ ] 监测范围有**示意图**，标明对应环号
- [ ] 监测项目表含：监测项目 / 监测形式（自动化 / 人工）/ 仪器类型 / 仪器精度 / 备注
- [ ] **附属结构、顶管通道**单独增加水平位移监测点
- [ ] **3.4 监测频率分阶段独立成表**（不能只用一张表概括全过程）
- [ ] 自动化监测频率**至少 6 次/天**（4 次/天会被打回——段塘大桥案例）
- [ ] 关键工序（顶进/穿越/钢板桩打拔等）频率加密说明
- [ ] **预估工作量表**：测点布设/拆除数量、监测次数、按阶段分项

## 四、控制网（第 4 章）

- [ ] **平面控制网示意图**（评审硬性要求——通途西路案例）
- [ ] 平面控制网**基准点位置明确**（含环号、车站段位置）
- [ ] **自动化全站仪对应环号**清楚标注
- [ ] 高程控制网明确**站内外工作基点具体位置**
- [ ] 高程工作基点**误差消除方法**说明（如沿用稳定历史基准点比对）
- [ ] **平面基准网复测**有专门小节，全过程统筹考虑
- [ ] 控制点选取与既有运营监测点的衔接关系说明

## 五、监测点布设（第 5 章）

- [ ] 监测点布设**断面图**（不只是平面图）
- [ ] 各类型监测点的**埋设方法+实景图**（道床沉降、结构沉降、水平位移、收敛）
- [ ] **静力水准仪布置图**及基准点位置（顶管/盾构必备）
- [ ] **附属及顶管通道水平位移监测点**单独增加
- [ ] 重点监测区**加密断面**明确表示
- [ ] **每阶段监测点布设平面图对应施工工况**（6 号线方案要求）
- [ ] 平面/高程**控制点位置**也标注在布点图上

## 六、监测方法（第 6 章）

- [ ] **自动化全站仪**：组网原理、数据采集、传输、参数
- [ ] **静力水准**：仪器型号、参数表（测量范围、最小读数、精度）
- [ ] **人工监测**：技术要求详细（精度等级、测段长度、闭合差）
- [ ] **数据辩伪**：粗差判别、误差消除方法
- [ ] **人工与自动化比测**章节（6 号线要求）
- [ ] 监测点受损后**修复及数据衔接**方法
- [ ] 完善各监测项目所采用的**监测方法、仪器型号及技术要求**（6 号线要求）

## 七、报警值与预消警（第 11 章）

- [ ] 报警值表分**预警值/报警值/控制值**三栏，加变化速率列
- [ ] **按施工阶段细化报警值**（专家评审常见加分项）
- [ ] 既有线**安评变形数值分析**引用并分解
- [ ] 报警/消警流程的**单项 vs 综合**预警区分
- [ ] **三色（黄/橙/红）响应流程图**完整
- [ ] **预警分级审核流程表**明确各方角色
- [ ] **消警流程**（连续 ≥3 日数据收敛）

## 八、应急与组织

- [ ] **应急响应流程图**
- [ ] **应急组织机构图**（地保办 / 地保车间 / 监测单位 / 施工 / 监理 / 设计 四级以上）
- [ ] **应急设备清单**
- [ ] **防汛防台专项**预案
- [ ] **24 小时应急联系电话**表
- [ ] **轨行区作业流程**与安全管理办法

## 九、附图附表（最常被打回项）

- [ ] 工程总平图（含新老结构 + 既有轨道位置）
- [ ] 与轨道交通结构关系图（剖面或三维示意）
- [ ] 平面/高程控制网示意图
- [ ] **按施工阶段分别**的监测点布设平面图
- [ ] 监测点布设断面图
- [ ] 静力水准仪布置图
- [ ] 应急/信息反馈流程图
- [ ] 测点位置统计表

## 十、其他

- [ ] **数据保密**章节（运营线路强制）
- [ ] **资料归档**说明（季度/年度报告、电子版）
- [ ] 与业主、施工、监理**配合服务**条款
- [ ] **参与建设各方联动机制**
- [ ] 章节编号无跳号，目录与正文一致
- [ ] 中文标点使用全角符号
- [ ] 所有规范名称用书名号《》
- [ ] 所有图表均有编号和题注（图 X.X-N、表 X.X-N）
