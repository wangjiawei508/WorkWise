## 1.3 监测目的
1. 对轨道交通运营安全性进行长期监测
2. 掌握所监测结构在长期运营中的状态
3. 掌握服役期内的荷载、环境作用及效应
4. 积累监测资料、提供危害治理可靠依据

## 1.4 监测内容与方法
### 1.4.1 监测内容
| 监测对象 | 监测项目 |
| --- | --- |
| 盾构区间道床/结构 | 沉降、收敛 |
| 类矩形/矩形盾构 | 沉降、收敛、净空断面 |
| 车站、明挖隧道、U 槽 | 沉降 |
| 联络通道（冷冻法 / 机械法） | 沉降、收敛、净空断面（机械法） |
| 道岔区 / 高架段 | 沉降、（视情况）水平位移 |
| 车辆段 / 停车场 / 附属用房 | 沉降 |
| 保护区建（构）筑物 | 沉降、（视情况）水平位移 |
| 全线 | 巡视巡查 |

### 1.4.2 监测方法
- 沉降：数字水准仪 + 铟钢条码尺，aBFFB
- 内径收敛：手持激光测距 / 全站仪
- 净空断面：三维激光扫描
- 水平位移：高精度全站仪 + 棱镜
- 巡视：目视 + 测缝针 / 裂缝仪
