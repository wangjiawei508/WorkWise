# 基准网联测与稳定性分析

运营期监测必须建立**"地面深桩控制网 → 站外工作基点 → 站台工作基点 → 测点"**的四级高程传递链。本文档描述该传递链的联测、平差与稳定性判定。

## 1 体系结构

```
深桩水准点（业主提供，每年 2 次成果）
        │ 一等水准检测
        ▼
站外工作基点（建（构）筑物深桩基础上）
        │ 一等水准联测
        ▼
站台工作基点（站台内开孔埋设）
        │ 监测Ⅱ级（中视附合）
        ▼
道床/结构沉降监测点
```

水平位移系统：

```
首级控制点（变形区外，间距 100m 左右）
        │ 多测回测角 + Ⅱ级测距
        ▼
工作基点（远端 20 m 外）
        │ 自由设站 / 后视定向
        ▼
监测点（棱镜 / 反射片）
```

## 2 高程基准与坐标系

- **高程基准**：原则采用 1985 国家高程基准（二期）；个别项目可能采用地方系，需在方案中固化并贯穿全周期。
- **平面基准**：水平位移一般用**相对坐标系**，沿隧道走向定 X 轴或 Y 轴，朝向基坑/敏感方向为"+"，远离为"-"。
- **数据衔接**：当业主提供的深桩成果发生改算时，新旧基准应做整体平差对比；改算量 >0.5 mm 时必须出"基准改正说明"。

## 3 深桩点检测（一等水准）

### 3.1 频率
- 业主每年上、下半年各提供一次成果
- 监测单位**全部测段**做一等水准检测
- 期次开工前若有路线明显变化（道路施工/沉降异常）即时复测

### 3.2 限差（GB/T 12897-2006）
| 项 | 限差 |
| --- | --- |
| 测段往返不符值 | 1.8√R mm |
| 环闭合差 | 2√F mm |
| 检测已测测段较差 | 3√R mm |

### 3.3 观测要求
- 数字水准仪 + 铟钢条码尺，aBFFB
- 三固定：固定仪器、固定人员、固定路线
- 每日开测前 i 角检校（费式法），≤15″
- 模式 = mean，每站观测 3 次取均值
- 转点尺垫脚踩实，间歇以偶数站结束于水准点（无法结束时选 3 个稳固固定点喷漆作标识并交接班）

## 4 上下联测（地面 ⇆ 地下）

将地面控制网高程通过出入口传到站台工作基点。

- **路线**：地面工作基点 → 出入口 → 站台工作基点（往返测）
- **等级**：国家一等水准
- **要点**：
  - 出入口段坡度大，控制视线长度 ≤ 30 m、前后视距差严控 ≤ 0.5 m
  - 楼梯/扶梯段尺垫易移动，必要时改用稳固固定点
  - 站台点观测优先在停运时段，避开列车/旅客振动

## 5 工作基点稳定性分析

### 5.1 判定流程
1. 当期联测完成后做严密平差，得到本期工作基点高程 H_i 与中误差 m_i
2. 与上期高程 H_{i-1}、中误差 m_{i-1} 比较，计算 ΔH = H_i − H_{i-1}
3. 联合中误差 m₀ = √(m_{i-1}² + m_i²)
4. 按下表判定：

| 序号 | 判定条件 | 稳定性 | 高程是否改正 |
| --- | --- | --- | --- |
| 1 | \|ΔH\| ≤ 2√2·m₀ | 稳定 | **不改正** |
| 2 | 2√2·m₀ < \|ΔH\| ≤ 2·2√2·m₀ | 较稳定 | 不改正 |
| 3 | 2·2√2·m₀ < \|ΔH\| ≤ 3·2√2·m₀ | 有可能沉降 | **改正** |
| 4 | \|ΔH\| > 3·2√2·m₀ | 有沉降 | **改正** |

> ΔH 正值表示上升，负值表示下沉。
> 该判定为**整网平差完成后**的判定；不得用单测段的高差较差直接代替。

### 5.2 改正方式
- 若工作基点判定为"沉降"，本期监测点高程应**统一改算**到稳定基点（站外深桩点）下
- 改正过程要在期报附录中给出推导公式与改正量表
- 工作基点稳定状态在期报中**单独成节**说明

## 6 平面基准网（水平位移用）

- 频率：1 次/月
- 流程：原始数据严密平差 → 两期坐标分量较差与 2m（m 为最弱点中误差）比较
- 判定：

| 较差 | 稳定性 | 处理 |
| --- | --- | --- |
| ≤ 2m | 稳定 | 不改正 |
| > 2m | 不稳定 | 弃用并改用上级控制点 |

- 控制网形式：环形或附合导线穿过测区时**需在测区内设置固定转点**
- 全站仪测量配合"自动照准 + 多测回测量"+ 测量云/标准平差软件；测前读取温度、气压做气象改正

## 7 控制网联测交底要点

- 出测前对所有作业人员书面交底：测段编号、视距上限、跑尺顺序、收尺规则
- 雨天/雷电/能见度低**停测**
- 高架段加强遮阳、防风（视线高度优先取下半视场）
- 控制网完成后由内业**当天**整理 → 隔日完成平差 → 一周内完成基准网联测报告

## 8 控制网联测报告必含章节

1. 项目概况、测段/路线分布图
2. 仪器与人员、检定证书复印件
3. 各测段往返不符值、环闭合差校核表
4. 整网平差结果（已知点/未知点/测段数 / 先验后验单位权中误差 / 最弱点中误差 / PVV / 自由度）
5. 工作基点稳定性判定表
6. 高程改正量与改正后成果表
7. 异常情况说明、整改/重测记录
8. 附件：原始记录、平差报告、现场照片

## 9 常见问题

- 深桩点成果未到，期次直接开工——禁止；至少要取得上半年/下半年成果之一
- 整网平差未做稳定性判别，直接套用上期高程——禁止
- 工作基点改算后未通知外业，导致期报抬头基准说明错误
- 上下联测视距差超限（出入口段）未重测，留作"补测"挂账
- 水平位移控制网未做气象改正
