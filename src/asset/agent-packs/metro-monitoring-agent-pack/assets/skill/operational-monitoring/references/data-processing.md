# 数据处理与精度评定

## 1 处理总流程

```
原始记录(.dat/.gsi/.raw)
    ├─ 1) 单站检查（仪器自检超限项）
    ├─ 2) 测段检查（视距差、累积、闭合差）
    ├─ 3) 数据导入平差软件（科傻 COSA / PowerAdj / StarNet）
    ├─ 4) 严密平差
    ├─ 5) 精度评定（先验/后验单位权中误差、最弱点中误差）
    ├─ 6) 基准点稳定性判定（见 baseline-network.md）
    ├─ 7) 测点高程/坐标提取
    ├─ 8) 变量计算（本期、累计、速率）
    ├─ 9) 成果表生成
    └─ 10) 自检（四道关）→ 内审 → 外审 → 出版
```

## 2 平差软件

### 2.1 软件选择
- **科傻 COSA / NASEW**：国内监测一等水准与监测Ⅱ级最常用
- **南方平差易**、**PowerAdj**
- **StarNet**（控制网平面/三维）

### 2.2 一等水准/Ⅱ级水准平差参数
- 选择"水准测量国标一等"或"监测Ⅱ级"模式
- 输入已知点（深桩点 / 站台工作基点）
- 输入观测高差与测段长度（km）
- 选择"严密平差（条件平差或间接平差）"

### 2.3 平差报告关键指标
| 指标 | 含义 | 验收 |
| --- | --- | --- |
| 已知点数、未知点数 | 控制网规模 | 报告必有 |
| 测段数 | 含中视测段单独标识 | 与外业一致 |
| 水准路线总长度 | km | 决定限差基准 |
| 先验单位权中误差 σ₀ | 设计精度，水准取 ±0.15 mm（监测Ⅱ级）或 ±0.45 mm（一等） | —— |
| 后验单位权中误差 μ | 验后实际精度 | 应 ≤ σ₀，否则查测站 |
| PVV | 改正数平方和 | 报告记录 |
| 自由度 | 多余观测 | ≥1 |
| 最弱点高程中误差 | 单点最差 | 应 ≤ 0.5 mm（Ⅱ级） / 0.3 mm（一等监测） |
| 最弱测段高差中误差 | 单测段最差 | 报告记录 |
| 闭合环数 | 环闭合差校核 | 各环列表 |

## 3 精度评定公式

### 3.1 偶然中误差（每千米）
$$M_\Delta = \pm \sqrt{\frac{1}{4n}\sum\frac{\Delta_i^2}{R_i}}$$

- Δ_i：测段往返高差不符值（mm）
- R_i：测段长度（km）
- n：测段数

### 3.2 全中误差（每千米）
$$M_W = \pm \sqrt{\frac{1}{N}\sum\frac{W_i^2}{F_i}}$$

- W_i：环闭合差（mm）
- F_i：环线长度（km）
- N：环数

### 3.3 验收
| 等级 | M_Δ 限差 | M_W 限差 |
| --- | --- | --- |
| 国家一等 | ±0.45 mm | ±1.0 mm |
| 监测Ⅱ级 | ±0.5 mm | ±1.0 mm |

## 4 变量计算

### 4.1 沉降量
- 本期变化量 ΔHₙ = Hₙ − Hₙ₋₁
- 累计变化量 ΣΔH = Hₙ − H₀（H₀ 为初始值）
- 沉降速率 v = ΔHₙ / 期间天数（mm/d）或 mm/月
- 正负号约定：**沉降为负、隆起为正**（在成果表抬头明示）

### 4.2 收敛量
- dSn = Sn − S(n−1)
- DSn = Sn − S0
- 净空扩张为正、收缩为负

### 4.3 水平位移
- ΔX = Xₙ − Xₙ₋₁
- ΔY = Yₙ − Yₙ₋₁
- 累计 ΣΔXY 同
- 速率按分量分别给出，矢量大小 = √(ΣΔX² + ΣΔY²)

### 4.4 工后变形量
工后 = 当期累计 − 通车交付期累计（"工后变形零点"按合同约定，通常取联调联试或正式运营前一次）

## 5 基准点稳定性判定（汇总）

> 完整流程见 `baseline-network.md`。

### 5.1 高程
- m₀ = √(m_{i-1}² + m_i²)
- \|ΔH\| 与 2√2·m₀、4√2·m₀、6√2·m₀ 三阈值比较，分四档（稳定 / 较稳定 / 有可能沉降 / 有沉降）
- 第三、第四档**改正本期监测点高程**

### 5.2 平面
- \|ΔXY\| 与 2m 比较（m = 复测最弱点中误差）

### 6.4 抬头声明
所有成果表表头必须包含：
- 项目名称
- 期次号、观测日期、签发日期
- 基准（高程基准、平面基准）
- 正负号约定
- 平差软件与版本
- 监测单位、主测人员、内审、外审

## 7 数据自检"四道关"

1. **数据检查**：限差是否满足
2. **仪器检查**：检定期内、状态正常
3. **测点检查**：测点是否被破坏、覆盖、迁移
4. **记录检查**：计算公式、成果表与原始记录一致；无人为改动

任一关失败立即返工，不得**带病提交**。

## 8 数据衔接

- 测点重测/迁移后累计值必须**从首次埋设开始累加**
- 原点号映射表保留至少 5 年
- 平差软件版本变更时，至少做一期"新旧版本对比"，结果差 >0.2 mm 时书面备案

## 9 常见 Pitfall

- 后验单位权中误差远小于先验（如 0.006 vs 0.095），可能是验前精度设置过宽——按规范取严的值复查
- 中视点测段单独评定时与附合主线测站精度混算
- 平差时已知点输入错误（取了上期未改正高程）
- 累计值未按"原点号衔接"，对换点导致曲线断点
- 速率按"自然月"还是"31 天"未统一，跨期对比失准
- 工后变形零点变更未在成果表抬头声明
