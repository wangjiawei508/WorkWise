# 沉降监测外业（一等水准 + 监测Ⅱ级中视附合）

## 1 适用对象
- 道床沉降（盾构、车站、明挖、U 型槽、出入段线）
- 结构沉降（管片、车站墙身、U 型槽侧壁）
- 联络通道沉降
- 道岔区、高架段沉降
- 建（构）筑物垂直位移

## 2 观测方法

### 2.1 总方法
"附合水准 + 中视水准"组合：
- 站台两个工作基点 → 附合水准两端
- 沉降监测点作为**中视点（转站点）**
- 左、右线监测点可同时施测
- 有联络通道的，水准路线必须**通过旁通道连接**，相邻两车站间的 4 个工作基点形成闭合水准网

### 2.2 单站规程
1. 后视站台基准点（A），读取后视 B 尺
2. 前视转站点，按 **aBFFB / "前-后-后-前"** 顺序观测
3. 本站结束后，**中视法**观测该区间所有沉降监测点
4. 进入下一站，重复 1–3，直至附合到另一站台点

奇数测站：**后-前-前-后**；偶数测站：**前-后-后-前**。每段往测/返测测站数均为偶数（起终点同尺）；由往返转换时两尺互换位置。

### 2.3 设站
连续测站上脚架两脚平行水准路线方向，第三脚轮换置于路线左/右侧。

### 2.4 测量模式
- 数字水准仪 = `mean`，每站 3 次取均值
- 振动明显地段随时增加测量次数
- 三固定（仪器/人员/路线）贯穿全期

## 3 限差速查

### 3.1 Ⅱ级水准观测主要技术要求
| 项 | 限差 |
| --- | --- |
| 视线长度 | ≤30 m |
| 前后视距差 | ≤0.5 m |
| 前后视距累计差 | ≤1.5 m |
| 视线离地面最低高度 | ≥0.3 m |
| 基、辅分划读数较差 | ≤0.3 mm |
| 基、辅分划所测高差较差 | ≤0.4 mm |

仪器自动报警时**立即重测本站**，不允许"强制保存"。

### 3.2 Ⅱ级垂直沉降监测控制网
| 项 | 限差 |
| --- | --- |
| 相邻基准点高差中误差 | ±0.5 mm |
| 测站高差中误差 | ±0.15 mm |
| 附合或环线闭合差 | ±0.3√n mm |
| 检测已测高差较差 | 0.4√n mm |

> n = 测站数。当含中视点时 n 取附合主线测站数，中视测段不计入闭合差但要单独评定测站精度。

## 4 仪器与配套

| 项 | 要求 |
| --- | --- |
| 仪器 | 数字水准仪（如 Trimble DiNi03），读数精度 0.01 mm，整套精度 ≤±0.3 mm/km 往返 |
| 标尺 | 2 m 铟钢条码尺，配套尺垫，检定证书有效 |
| 配件 | 尺灯、电池、脚架、对讲机、记录板 |
| 检校 | 每日开测前 i 角检校；运输颠簸后增校 |

## 5 外业准备清单

1. 检查仪器、电池、尺灯、对讲机、雨靴、口罩
2. 当日 i 角检校记录归入日报
3. 测段卡片：始终点点号、上期高程、当期理论里程、限差预计算
4. 检查测点是否清晰、是否被覆盖（道床清扫）
5. 与运营调度沟通"天窗点"作业时间
6. 危险源识别（接触网、列车进路、消防设备）

## 6 单条作业线流程

```
日 i 角检校 → 出测 → 站台 A 点架站 → aBFFB 后视/前视 → 中视采点
→ 转站到下一断面 → 重复 → 附合至站台 B 点 → 检查闭合差
→ 立即导出数据 → 现场粗校 → 异常立即重测 → 回办公室提交
```

> 当日数据**当日导出归档**，由主测人员负责，禁止延至次日。

## 7 中视法常见 Pitfall

- 中视点放在视距 >30 m 处——视距超限即超精度
- 漏点（特别是浮置板边界、联络通道交界）
- 转点位置和中视点位置混淆，导致点号串位
- 中视记录段过长、过期次衔接断裂
- 站台大理石盖板观测完未复原，遭运营投诉

## 8 雨季/夏季/振动地段的额外措施

- 振动明显（变电站、风井、人流密集站台）：增加观测次数至 5 次
- 雨季能见度差：检查尺灯供电、增加视线高余量
- 高温膨胀：尺面温度记录入手簿
- 列车通过 30 秒内禁测，等待振动消失

## 9 安全要点

- 严格遵守业主"天窗点"作业制度（一般夜间 0:00–4:00）
- 进入轨行区按规定佩戴反光衣、口罩、安全帽，需轨行区准入证
- 接触网区作业前确认是否已停电挂地线，未挂地不上长杆
- 多组同时作业时保持 ≥50 m 间距，避免相邻仪器干扰
- 巡视发现明显异常（裂缝、渗漏、错台）立即上报项目负责人，按 `warning-and-disposal.md` 处置

## 10 单期作业产出物（外业部分）

1. 原始数据文件（.dat / .gsi / .raw），按"区间-上下行-日期-主测"命名
2. i 角检验记录
3. 当日观测日志（点号清单、异常说明、天气、温度、起止时间）
4. 现场照片（必要时）
5. 数据交接登记（外业→内业）
