# 水平位移监测（全站仪法）

## 1 适用对象
- 轨行区车站、隧道结构水平位移
- 高架段、U 型槽侧壁水平位移
- 保护区周边建（构）筑物水平位移
- 基坑抽测（轨行区附近基坑施工期间影响监测）

## 2 坐标系建立

### 2.1 平面基准
- 一般采用**相对坐标系**，沿隧道走向定 X 轴或 Y 轴
- 朝向基坑/敏感方向位移为"+"，远离为"-"
- 若基坑边线与隧道方向不平行，需按"夹角"做位移量分解

### 2.2 首期坐标建立
1. 在测区内选 2 个通视的道床点作为基准
   - 测区较短且通视良好：尽量取测区两端
   - 测区较长：尽量在测区中段选点，距离 200 m 左右
2. 以其中一点为架站点，**假定坐标**（例：X=10000, Y=50000）
3. 用方位角 0°/90°/270° 后视另一点设站
4. 设站完成后**重新观测**另一点，以测得坐标作为该点最终值
5. 相对坐标系建立完毕，向两端引测确定首级控制点

### 2.3 引测原则
- 沿通视方向加临时转点
- 转点之间边长 ≤ 基准边，且**大致相等**
- 引测方法：**多测回测角 + Ⅱ级测距**

## 3 测点布设

### 3.1 工作基点
- 远离测区 ≥20 m
- 采用**对中架**或直接植入棱镜杆套（莱卡圆棱镜）
- 必要时加保护盒

### 3.2 首级控制点
- 布设于变形影响区域外，距工作基点 ≈100 m
- 形式同工作基点
- 与工作基点之间设固定转点（利用原有道床点或对中架）；穿过测区的附合导线必须在区内设固定转点

### 3.3 监测点
- 与沉降监测点处于同一断面
- 钻孔 → 植入膨胀螺丝 → 安装莱卡小棱镜或同规格反射器
- 棱镜方向朝向作业方向，便于自动照准

## 4 仪器与精度

| 项 | 要求（监测Ⅱ级） |
| --- | --- |
| 全站仪 | ≥ ±2″、2+2ppm |
| 水平角测回数 | 9（首期）/ 3（复测） |
| 距离测回数 | 3 |
| 半测回归零差 | ≤6″（Ⅰ级仪器） |
| 一测回 2C 互差 | ≤9″ |
| 同方向各测回较差 | ≤6″ |
| 一测回读数间较差 | ≤3 mm |
| 单程各测回较差 | ≤4 mm |
| 往返/不同时段较差 | ≤2(a+b·D) |

### 4.1 控制网 Ⅱ级技术指标
| 项 | 限差 |
| --- | --- |
| 相邻基准点点位中误差 | ±3.0 mm |
| 平均边长 | 150 m |
| 测角中误差 | ±1.8″ |
| 最弱边相对中误差 | ≤1/70000 |

## 5 观测方法

### 5.1 出测前
- 仪器气象改正：读取**温度、气压**
- 棱镜常数、加常数校验
- 设置各项限差，超限即时重测
- 配合"测量云 / 同等多测回测量软件"进行自动照准、自动测回

### 5.2 测回
- 多测回测角：盘左盘右各 1 次为 1 测回
- 距离测量与角度测量同时进行
- 自动测量软件在每测回完成后给出超限提示

### 5.3 时段
- 优先夜间天窗作业，避免高温、大温差引起仪器漂移
- 同一控制网外业应**当夜完成**，避免跨夜数据合并

## 6 基准点稳定性分析

按 1 次/月频率复测平面基准网：
- 严密平差后对比**两期同点的 (Δx, Δy)**
- 较差与 2m（m = 复测控制点最弱点点位中误差）比较

| 较差 \|ΔXY\| | 稳定性 | 处理 |
| --- | --- | --- |
| ≤ 2m | 稳定 | 不改正 |
| > 2m, ≤ 3m | 较稳定 | 关注 |
| > 3m | 不稳定 | 启用上级首级控制点 |

## 7 数据处理（要点，详见 `data-processing.md`）

- 严密平差（科傻、PowerAdj、StarNet 等）
- 平差报告应含：单位权中误差（先验/后验）、最弱点位中误差、最弱边相对中误差、PVV、自由度
- 监测点本期成果 = 当期坐标，本期变化量 = 当期 − 上期，累计 = 当期 − 首次
- **变化量分量**：ΔX、ΔY，按坐标系约定输出符号
- 输出表头注明"测距加常数、温度、气压、棱镜类型"

## 8 限差红线

- 半测回归零差超限即重测
- 一测回 2C 互差超限即重测
- 任意一项气象改正未做不得提交成果
- 后视点稳定性未判定即用，本期成果作废

## 9 常见 Pitfall

- 假定坐标系方向定反，导致符号约定与方案不一致
- 多测回测量软件未开启限差监控，超限值进入平差
- 工作基点对中架松动未发现，全期累计误差累加
- 棱镜小棱镜与莱卡圆棱镜混用，加常数错误
- 温度/气压未读取，长测线产生 ppm 级偏差
- 自动照准误锁邻近反光物（道钉、警示标志）

## 10 单期产出

1. 控制网原始数据（含每测回明细）
2. 控制网平差报告（含稳定性判定）
3. 监测点坐标成果表（X、Y、ΔX、ΔY、累计、备注）
4. 变形曲线图（按断面 / 按时间）
5. 异常说明、复测记录
