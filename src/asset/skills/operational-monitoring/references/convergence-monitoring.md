# 收敛监测与三维激光扫描

收敛监测有两类对象：
- **内径收敛**——监测圆形/类矩形盾构隧道在某断面上两点之间净距的变化；常用方法为手持激光测距、全站仪精测。
- **净空断面**——监测整个隧道内壁的几何形状（断面）变化；要求用**三维激光扫描仪**完成。

## 1 内径收敛（手持激光测距法）

### 1.1 仪器
| 项 | 要求 |
| --- | --- |
| 激光测距仪 | 标称精度 ±1 mm |
| 检定 | 业主/监理报验通过 |
| 配件 | 长杆（盖印用）、绝缘杆（接触网区）、印章 |

### 1.2 测点
- 断面间距：**5 环（约 6 m）**
- 每断面 2 条测线：**0°–180°（竖向）**、**90°–270°（横向）**
- 顶部测点：方形十字印章，按 1.2 中的"长杆 + 绝缘杆"方式盖印
- 底部测点：与道床沉降点共用（道床面）

### 1.3 观测方法
1. 测距仪紧贴测线一端（O-1），对准另一端（O-2）
2. 读数取 3 次独立观测中数；中数与单次读数较差应 ≤ ±2 mm
3. 同一测线**初始值需独立观测 3 次**取中数

### 1.4 数据计算
- 第 n 期净空变化：dSn = Sn − S(n−1)
- 第 n 期累计变化：DSn = Sn − S0
- 正负号：扩张为正、收缩为负（在表头注明）

### 1.5 接触网区作业
- 必须先申请**接触网停电挂地线**
- 由供电专业操作人员实施挂地，两端确认后才开始作业
- 长杆必须为**绝缘杆**

## 2 内径收敛（全站仪法）

适用于联络通道、机械法盾构、需要更高精度（≤±1 mm）或长测线（>15 m）的场景。

### 2.1 仪器与精度
- 高精度全站仪：测角 ±1″–±2″、测距 1+1ppm 至 2+2ppm
- 反射片或棱镜
- 与项目方案约定限差

### 2.2 观测要点
- 自由设站于断面外（避免遮挡测线）
- 后视已知道床点 / 工作基点定向
- 每条测线**3 测回**，测回间较差 ≤ 1 mm
- 取测距与坐标反算两套结果对比

### 2.3 联络通道（机械法）收敛
- 每条通道按断面间距 5 环布点
- 通道中点必布断面
- 通道与正线交界处必布断面
- 0°/90°/45° 三条测线（视通道形状定）

## 3 净空断面三维激光扫描

### 3.1 适用范围
- 类矩形盾构净空断面
- 矩形盾构净空断面
- 机械法/盾构法联络通道净空断面
- 普通圆形盾构的"高精度断面变形"补充监测

### 3.2 仪器与精度
| 项 | 推荐参数 |
| --- | --- |
| 扫描仪 | 站式三维激光扫描仪（如 FAROFocusS Plus 350） |
| 测距噪声 | ≤0.3 mm @ 10 m |
| 角度精度 | ±19″ |
| 扫描分辨率 | 6 mm @ 10 m 或更密 |
| 配套靶标 | 球形靶标 / 棋盘格靶标 |
| 拼接残差 | ≤2 mm |

### 3.3 外业流程
1. **测站布设**：沿隧道走向每 10–20 m 设 1 站；相邻站重叠区段应包含 ≥3 个公共靶标
2. **靶标布设**：每站至少 3 个球标 / 棋盘格，靶标分布"三角包围"测站，远近高低各异
3. **公共控制**：靶标坐标由全站仪联测工作基点而获得；同一线段所有测站靶标在同一坐标系
4. **扫描**：分辨率/质量按方案设定；列车通过时停扫
5. **现场快查**：每站完成立即查看噪声、覆盖率、靶标识别率
6. **回办公室拼接**

### 3.4 点云拼接
- 优先**靶标约束拼接**（同一坐标系下，残差 ≤2 mm）
- 辅以**ICP 几何拼接**（仅在靶标识别失败时使用）
- 拼接报告必须包含每站残差、整体 RMS

### 3.5 断面提取与拟合
- 沿设计中心线按设定间距（如 0.5 m）切片
- 每个断面进行**椭圆 / 多段线拟合**
- 拟合粗差按**拉依达准则（3σ）**剔除，超 1% 点参与剔除时复扫
- 输出断面：长短轴、椭圆度、与设计断面比较的偏差云图

### 3.6 数据成果
- 隧道净空断面对比图（本期 vs 初始 / 上期）
- 椭圆度时序曲线
- 关键断面 0°–180°、90°–270° 收敛值（与全站仪/激光测距对比）
- 整段最大变形点位置

## 4 数据处理（收敛通用）

### 4.1 差异数据检查（按四道关）
1. **数据检查**：误差是否在限差以内
2. **仪器检查**：检定期内、状态正常
3. **测点检查**：测点是否被破坏、覆盖
4. **记录检查**：计算公式、成果表与原始记录是否一致；有无人为改动

### 4.2 责任追溯
- 一次：通报批评 / 口头教育
- 二次：罚款或辞退（主责）/ 通报批评（非主责）
- 管理人员失察：通报、罚款或辞退

### 4.3 数据衔接
重测点累计值**必须从首次埋设开始累加**，方案保留原点号映射表。

## 5 限差与红线

| 项 | 限差 |
| --- | --- |
| 单次激光测距与中数较差 | ≤±2 mm |
| 全站仪 3 测回较差 | ≤1 mm |
| 三维激光扫描站间拼接残差 | ≤2 mm |
| 椭圆拟合粗差剔除 | 拉依达 3σ |
| 同一断面"三维激光"与"激光测距"互校 | ≤±3 mm |

## 6 常见 Pitfall

- 测距仪未紧贴印章对中标志，引入 1–3 mm 偶然差
- 顶部印章盖偏（长杆角度未控好）
- 接触网区未停电挂地直接作业（重大安全违规）
- 三维激光扫描站间未布足靶标，靠 ICP 强拼接，残差 >5 mm
- 列车通过未停扫，单站噪声放大数倍
- 断面拟合不剔粗差，导致最大变形点误判
