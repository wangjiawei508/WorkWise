# 期次归档自检清单

> 用途：每期次归档前由内业责任人逐项确认。任一项未达标不得提交期报。

## 一、原始数据
- [ ] 数字水准 `.dat` / `.gsi` 全部入库（按测段/区间齐全，无缺漏）
- [ ] 全站仪原始观测文件入库（如有水平位移）
- [ ] 三维激光扫描原始站点云入库（如有）
- [ ] 数据 SHA-256 校验文件生成

## 二、仪器与检校
- [ ] i 角检验记录入档（每个外业日一份）
- [ ] 仪器检定证书在有效期内
- [ ] 仪器维护记录更新

## 三、控制网联测
- [ ] 控制网联测报告完成
- [ ] 各测段往返不符值、环闭合差校核合格
- [ ] 整网平差报告完整（含先验/后验单位权中误差、最弱点、PVV、自由度）
- [ ] 工作基点稳定性判定明确（按 |ΔH| 与 2√2·m₀ 阈值四档分类）
- [ ] 改正机制（若有）说明清楚

## 四、平差与成果
- [ ] 沉降成果表完成、抬头基准/正负号/签名齐全
- [ ] 收敛成果表完成（如有）
- [ ] 水平位移成果表完成（如有）
- [ ] 平差工程文件保留（可重新计算）
- [ ] 文字报告与成果表数值一致

## 五、巡视巡查
- [ ] 巡视报表含里程坐标
- [ ] 照片包含 GPS / 位置信息或里程标注
- [ ] 异常事件均有处置说明

## 六、异常处置（如有）
- [ ] 预警快报正本归档
- [ ] 业主签收回执归档
- [ ] 加密观测记录归档
- [ ] 消警备案文件（若已闭环）

## 七、报送
- [ ] 期监测报告内审通过
- [ ] 外审（监理）通过
- [ ] 业主签收回执留存
- [ ] 电子档案上传至档案系统（如有）

## 八、命名与归位
- [ ] 文件命名符合 `<项目编号>_<期次>_<类别>_<上下行/范围>_<版本>.<扩展>` 规范
- [ ] 文件已归至对应 13 大类子目录
- [ ] `YYYY-MM-期次号_<类别>` 文件夹结构正确
