# 商务标模块（资信标 / 商务响应）

技术标之外，投标文件通常还含**商务标（资信/商务响应）**与**报价文件**。本模块补齐商务标编制；报价文件（价格分）由人工另行核算，工程类报价可参考 `references/engineering-bid.md` 的收费标准工作流。

## 一、商务标常见组成

按招标文件"投标文件组成/装订顺序"为准，常见如下：

| 序号 | 内容 | 说明 | 证据/占位 |
| --- | --- | --- | --- |
| 1 | 投标函及投标函附录 | 法定格式，逐字套招标模板 | `{公司名称}` `{投标报价}` `{工期}` `{法定代表人}` |
| 2 | 法定代表人身份证明 / 授权委托书 | 授权代理人时必附 | `{被授权人}` `{委托事项}` |
| 3 | 投标保证金凭证 | 银行回单/保函 | `[资质占位符] 保证金缴纳凭证` |
| 4 | 营业执照 | 三证合一副本 | `[资质占位符] 营业执照` |
| 5 | 资质证书 | 对应招标要求的行业/专业资质 | `[资质占位符] {资质名}` |
| 6 | 财务状况 | 近 N 年审计报告/资产负债表 | `[资质占位符] 审计报告` |
| 7 | 依法缴纳税收社保证明 | 完税凭证、社保缴纳证明 | `[资质占位符] 完税/社保证明` |
| 8 | 类似项目业绩 | 合同+中标通知+验收/用户报告 | `{案例项目名}` + `[资质占位符] 合同扫描件` |
| 9 | 项目团队 | 项目经理/技术负责人及主要人员证书、社保 | `{项目经理姓名}` + `[资质占位符] 证书` |
| 10 | 信誉证明 | 无重大违法、无失信、获奖、荣誉 | `[资质占位符] 信用中国截图` |
| 11 | 商务偏离表 | 逐条应答商务条款（工期、付款、质保、服务） | 见下 |
| 12 | 其他承诺函 | 廉洁、保密、售后、履约等 | 套模板 |

## 二、资格审查符合性对照（先做，防废标）

从招标"资格要求/资格审查表"逐条抽出，做成对照表，先自检是否满足，任一不满足即高危：

| 序号 | 资格要求(原文) | 我方情况 | 佐证材料 | 满足情况(人工) |
| --- | --- | --- | --- | --- |
| 1 | 具备XX资质 | | | 满足/不满足 |
| 2 | 近3年类似业绩≥N个 | | | |
| 3 | 项目经理持XX证书且社保在本单位 | | | |

⚠️ 资格审查是**通过性（合格/不合格）**判断，不满足直接出局，比技术分更优先核查。

## 三、商务偏离表（核心应答表）

对招标"合同主要条款/商务要求"逐条应答，满足情况使用“完全满足 / 满足 / 部分满足 / 不满足”四档并如实填写。出现部分满足或不满足时，必须标红提示偏离风险并交由用户决定是否整改、澄清或放弃投标；严禁为了看起来合规而把真实偏离改写成“满足”：

| 序号 | 招标商务条款(原文) | 我方响应 | 偏离(无/正/负) | 说明 |
| --- | --- | --- | --- | --- |
| 1 | 工期120日历天 | 承诺110日历天完成 | 正偏离 | 优于要求 |
| 2 | 质保期2年 | 提供2年免费质保 | 无偏离 | 完全满足 |
| 3 | 付款：验收后30日 | 完全接受 | 无偏离 | |

## 四、业绩与团队的证据规则

- **业绩三件套**：合同 + 中标通知书 + 验收/用户满意度报告，缺一评委可能不认；业绩表列"项目名/业主/金额/规模/时间/内容/联系人"，全部用占位符与"详见附件"引用。
- **人员证据**：证书 + 近N月社保记录 + 劳动合同/在职证明；项目经理常要求"同时只任一个在建标段"，注意冲突。
- **不编造**：业绩、金额、证书编号、社保一律占位或引用附件，由用户填真实值。

## 五、商务标质检要点（并入步骤⑦）

- 资格审查每条是否满足、佐证是否齐全（高危）；
- 投标函报价/工期/有效期与商务偏离表、报价文件是否**三处一致**（不一致=废标风险）；
- 保证金金额/形式/递交时间是否符合；
- 签字盖章位置是否齐全（投标函、授权书、每页骑缝等按招标要求）；
- 承诺函是否覆盖招标要求的全部承诺事项；
- 装订顺序/份数（正本副本/电子件）/密封是否符合"投标人须知"。

## 六、与技术标共用的原则

同样遵守：事实只来自招标文件、补充/澄清文件和用户提供且能够核验的企业证据；不编造；工作稿占位符在终稿前清零；内部风险标签不混入正文承诺；每条要求逐条响应且可追溯。
