# 行业细分关注点（步骤①.5 辅助）

判定标的四模式后，再按行业微调评分关注点、必备章节与证据。招标含明确行业特征词时套用；若无对应指南，可先联网检索该行业的投标结构要求、关键要点、评分关注点，再补充本文件。

## 行业判断关键词

| 行业 | 关键词 | 常见模式 |
| --- | --- | --- |
| IT/信息化 | 信息化、软件、系统集成、数据中心、云计算、大数据、人工智能、数字化、智慧XX | software-platform / hybrid |
| 建筑工程 | 建筑、施工、工程、装修、市政、道路、桥梁、隧道 | 工程标（另配流程）/ hybrid |
| 医疗健康 | 医疗、健康、医院、设备、药品、诊断、治疗、康复 | goods / hybrid |
| 教育服务 | 教育、学校、教学、培训、校园、课程、实验室、图书馆 | service / hybrid |
| 制造业 | 制造、设备、自动化、智能制造、工业、生产线、供应链、质量 | goods / hybrid |
| 物流运输 | 物流、运输、仓储、配送、供应链、快递、货运、冷链 | service / hybrid |
| 咨询服务 | 咨询、管理、审计、法律、评估、规划、研究、策划 | service |

## 各行业侧重

- **IT/信息化**：总体架构、功能模块、性能指标、安全等保/密评/信创、数据/接口、运维 SLA、类似系统案例。
- **建筑工程**：资质等级、项目经理/建造师、施工组织设计、工期与进度、安全文明施工、质量保证体系、机械设备。（施工组织设计需工程标流程）
- **医疗健康**：产品注册证/医疗器械资质、参数响应表与偏离表、临床/检测报告、售后与培训、供货安装调试。
- **教育服务**：教学场景理解、师资/团队、课程/培训方案、设备配套、验收与运维、案例业绩。
- **制造业**：设备参数与产能、工艺/自动化方案、质量体系、交付安装调试、供应链保障、售后备件。
- **物流运输**：服务网络与时效、SLA/KPI、人员车辆配置、信息化系统、应急预案、绩效考核。
- **咨询服务**：方法论、团队资历与投入、工作计划与交付物、质量控制、成果验收、类似案例。

## 内容通用红线
- 严格按评分标准编写，不泛泛而谈；每小节 ≥3 段，每章配表格；
- 禁用"我方/我们"，用"将/项目组"替代；
- 除非招标明确要求，正文不写金额/预算；
- 可验证事实（资质/案例/参数/人员）用占位符或引用附件，不编造。
