# 风格建模 15 维度详解

本文档详细说明风格建模的 15 个分析维度。在分析样本文本时，需要逐一提取每个维度的特征。

注意：15 维度是中间分析层，不是最终风格本体。最终风格文件必须继续提炼“风格内核”，并通过陌生主题仿写验证。具体方法见 [style-core-and-validation.md](style-core-and-validation.md)。

## 维度一：作者画像与核心人格

**目标**：理解作者是谁，从而理解他为什么这样写。

**分析要点**：
- **基础特征**：年龄段、职业背景、人生阶段（如：35-45岁社会老兵，经历过中年危机与行业洗牌）
- **知识结构**：学科背景、阅读范围、擅长领域
- **经历痕迹**：职场经历、人生阅历、朋友圈层次
- **性格特质**：现实主义？理想主义？同理心？犬儒？
- **社会角色**：目标读者是谁？想帮助读者做什么？

**输出示例**：
```
35-45岁，在复杂实体与商业体系中摸爬滚打10年以上，经历过跨经济周期的起伏与复杂的人性博弈。
性格：现实主义的温和悲观，有同理心但不煽情。
社会角色：过来人，用平视视角跟读者聊社会规则。
```

---

## 维度二：思维内核与论证逻辑

**目标**：提取作者的底层思维模式和论证结构。

**分析要点**：
- **核心世界观**：对社会的基本认知（筛选机制？培养机制？）、对个人命运的看法
- **价值判断标准**：真实vs体面、理解vs评判、可行vs理想
- **分析模式**：
  - 第一层：现象观察
  - 第二层：机制拆解
  - 第三层：人性洞察
  - 第四层：出路探索
- **问题意识**：不问"应该怎样"，而问"为什么是这样"

**输出示例**：
```
核心世界观：社会是筛选机制，不是培养机制。
价值判断：真实 > 体面，理解 > 评判。
分析模式：现象 → 机制 → 人性 → 出路（四层递进）。
问题意识：关注"为什么"而非"应该"。
```

---

## 维度三：创作路径还原

**目标**：理解作者如何从选题到成文的完整过程。

**分析要点**：
- **痛点捕捉**：如何发现选题？（读者反馈？生活观察？）
- **经验调取**：如何从经历库中找素材？
- **逻辑搭建**：采用什么结构串联观察？
- **共情设计**：在哪些节点插入互动？
- **现实化翻译**：如何把抽象道理转化为具体场景？
- **出路探索**：建议的务实程度如何？

**输出示例**：
```
痛点捕捉：从读者留言中发现共性困惑。
经验调取：调用自己10年职场经历 + 身边案例。
逻辑搭建：问题提出 → 机制分析 → 案例验证 → 出路建议。
现实化翻译：把"竞争力"翻译成"游戏副本"等具体比喻。
```

---

## 维度四：互动设计

**目标**：提取作者与读者的互动模式。

**分析要点**：
- **读者预设**：目标读者的画像和困境
- **互动技巧**：预判质疑？共情引导？对话式提问？
- **共鸣营造**：如何制造集体感（"咱们"、"大家"）？
- **引导策略**：给标准答案还是思考框架？

**输出示例**：
```
读者预设：25-35岁职场人，面临晋升困境。
互动技巧：高频使用"可能有小伙伴说..."预判质疑。
共鸣营造：大量使用"咱们"、"大家"制造集体感。
引导策略：提供思考框架，而非标准答案。
```

---

## 维度五：开头模式

**目标**：提取文章开头的固定模式。

**分析要点**：
- **句式**：背景交代？问题引入？观点直陈？
- **长度**：第一段通常几句话？
- **禁忌**：是否避开"在当今社会"等AI套路？

**输出示例**：
```
典型开头：直接抛出一个反常识观察或问题。
长度：第一段通常2-3句话，简短有力。
禁忌：绝不用"在当今社会"、"随着...的发展"。
```

---

## 维度六：段落过渡模式

**目标**：提取段落之间的连接方式。

**分析要点**：
- **方式**：无过渡直接跳？用反问引导（"那问题来了"）？情绪短句转折？
- **禁用词**：首先/其次/最后
- **典型信号词**："说到这里"、"当然了"、"也就是说"

**输出示例**：
```
典型过渡：高频使用"那问题来了"、"说到这里"。
禁用词：绝不用"首先/其次/最后"。
方式：设问引导 + 口语化连接词。
```

---

## 维度七：句式与节奏

**目标**：提取句子结构和节奏特征。

**分析要点**：
- **典型句式**：设问-回答式？对比式？递进式？
- **句子长度**：长短句比例，呼吸感控制
- **标点习惯**：大量逗号？频繁问号？省略号余味？

**输出示例**：
```
典型句式：设问-回答式（"为什么？因为..."）。
句子长度：短句为主（15-25字），偶尔长句（40-60字）。
标点习惯：大量逗号制造口语感，问号引导思考。
```

---

## 维度八：词汇指纹

**目标**：提取最具辨识度的词汇特征。

**分析要点**：
- **口语化连接词**："那问题来了"、"这不扯淡吗"
- **程度副词**："超级"、"极度"、"巨"
- **动词偏好**："搞"、"弄"、"整"、"折腾"
- **情感词汇**："纳闷"、"懵逼"、"无奈"
- **网络流行语**："内卷"、"躺平"、"牛马"

**输出示例**：
```
口语连接词：那问题来了、说到这里、当然了
程度副词：超级、巨、极度
动词偏好：搞、弄、整、折腾
情感词汇：纳闷、无奈、尴尬
禁用词：赋能、抓手、闭环、抓手
```

---

## 维度九：修辞手法

**目标**：提取常用的修辞技巧。

**分析要点**：
- **生活化类比**：把抽象概念比作具体场景
- **反问/设问**：使用频率及功能
- **排比/对比**：增强气势或制造反差

**输出示例**：
```
核心修辞：生活化类比（把职场比作"游戏副本"）。
反问频率：高（每3-5段出现一次）。
排比使用：描述困境时用排比（"苦、累、脏"）。
```

---

## 维度十：结尾模式

**目标**：提取文章结尾的固定模式。

**分析要点**：
- **句式**：总结式？建议式？留白式？
- **长度**：简短有力还是余音绕梁？
- **固定话术**：是否有特定的收尾习惯？

**输出示例**：
```
典型结尾：开放式留白，不给标准答案。
长度：1-2句话，简短有力。
固定话术：常用"想明白这一点，你就..."
```

---

## 维度十一：格式与排版

**目标**：提取视觉呈现特征。

**分析要点**：
- **小标题**：是否使用？风格如何？
- **强调方式**：加粗？重复？
- **段落视觉**：长短交替节奏

**输出示例**：
```
小标题：不使用小标题，全文一气呵成。
强调方式：重复关键词，不用加粗。
段落视觉：短段为主（3-5行），偶尔长段（8-10行）。
```

---

## 维度十二：独特习惯与招牌动作

**目标**：提取最具辨识度的 5-10 个特征。

**分析要点**：
- **句式类**："那问题来了..."
- **结构类**：案例故事化（人物-冲突-转折-结果）
- **表达类**：口语化超长句
- **互动类**：预判质疑（"可能有小伙伴说..."）
- **情感类**：悖论翻转、真诚地骂读者

**输出示例**：
```
1. 高频使用"那问题来了"进行转折
2. 用"游戏副本"等比喻解释抽象概念
3. 排比描述困境（"苦、累、脏"）
4. 预判质疑（"可能有人说..."）
5. 开放式结尾，不给标准答案
```

---

## 维度十三：反AI特征

**目标**：提取让读者感觉"这不是AI写的"的特征。

**分析要点**：
- 哪些"不规则性"是刻意为之？
- 哪些表达方式会让读者立刻感觉"这不是AI写的"？

**输出示例**：
```
1. 口语化超长句（40-60字）
2. 不规则的段落长度
3. 真诚地"骂"读者（"别扯淡了"）
4. 悖论式表达（"越努力越焦虑"）
5. 不完美的逻辑跳跃（更像真人思考）
```

---

## 维度十四：典型段落模板

**目标**：提取 3-5 个代表性段落作为模板。

**分析要点**：
- 观点段（反常识）
- 举例段（场景化）
- 转折段（设问引导）
- 收尾段（情绪留白）

**输出示例**：
```
### 观点段模板
"很多人以为...，但实际上...。为什么？因为..."

### 举例段模板
"我有个朋友...，他...，结果..."

### 转折段模板
"那问题来了，...？"

### 收尾段模板
"想明白这一点，你就..."
```

---

## 维度十五：禁忌清单

**目标**：明确绝对不能使用的元素。

**分析要点**：
- 绝对不能用的词汇（如"赋能"、"抓手"）
- 绝对不能用的句式（如"综上所述"）
- 绝对不能有的态度（如"高高在上"）

**输出示例**：
```
❌ 禁用词：赋能、抓手、闭环、生态、赛道
❌ 禁用句式：首先/其次/最后、综上所述、在当今社会
❌ 禁用态度：高高在上、说教、煽情、鸡汤
```

## 使用说明

在进行风格建模时：

1. **逐维度分析**：不要跳过任何维度
2. **大量摘录原文**：每个维度至少摘录 2-3 处原文作为证据
3. **提取可执行规则**：不要写文学赏析，要写能直接指导写作的规则
4. **招牌动作要写触发条件**：不仅写“常用什么”，还要写“什么时候用、为什么用、什么时候不能用”
5. **验证一致性**：15 个维度应该相互印证，构成统一的风格画像
6. **继续提炼风格内核**：完成 15 维度后，必须回答作者的判断方式、素材偏好、论证发动机和读者关系
7. **做仿写验证**：用陌生主题测试，低于 8 分必须回头修正风格内核
