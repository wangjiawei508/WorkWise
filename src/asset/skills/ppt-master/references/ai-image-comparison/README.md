# AI image comparison gallery

The upstream PPT Master project includes a visual comparison gallery for image palettes, renderings, and diagram types. WorkWise intentionally omits the generated comparison PNG files to keep the desktop installers small.

The text catalogs in `image-palettes/`, `image-renderings/`, and `image-type-templates/` remain available. When a user needs a visual comparison, generate or review images inside the current workspace instead of relying on bundled gallery assets.
