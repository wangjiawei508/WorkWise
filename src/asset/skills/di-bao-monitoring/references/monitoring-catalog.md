# 监测项、点号与阈值参考

本文件用于识别地保监测项目、点号规则、常见频率、正负号和报警层级。这里的内容是通用经验，必须服从当前项目的监测方案、安评报告、专家意见和运营单位要求。

## 监测对象与常见监测项

| 场景 | 常见监测项 |
|---|---|
| 地下盾构区间 | 道床沉降、结构沉降、水平位移、水平收敛、竖向收敛、远端沉降、巡视 |
| 车站及附属结构 | 结构沉降、道床沉降、水平位移、收敛、巡视 |
| 邻近基坑 | 自动化沉降/位移、人工复核、远端沉降、施工工况巡视 |
| 盾构/顶管穿越 | 沉降、水平位移、收敛、静力水准、振动、轨道几何 |
| 高架/桥梁/出入场线 | 桥墩沉降、桥墩水平位移、桥墩倾斜、盖梁/梁端/承台沉降、桥面沉降、桥面道床沉降、桥面结构沉降、巡视 |
| 控制网 | 平面控制网、高程控制网、工作基点复测和成果更新 |

## 点号识别规则

点号因项目而异，只能作为识别线索，不能作为唯一依据。

| 点号形式 | 常见含义 |
|---|---|
| `S*` / `X*` | 上行线 / 下行线 |
| 含 `z` | 自动化监测点 |
| `SD`、`XD` | 道床沉降，常见为人工或项目自定义 |
| `SDz`、`XDz` | 道床沉降，常见为自动化 |
| `SJz`、`XJz` | 结构沉降，常见为自动化 |
| `Sw`、`Xw` | 水平位移，常见为人工 |
| `Swz`、`Xwz` | 水平位移，常见为自动化 |
| `SYD`、`XYD` | 远端沉降 |
| `ZSJ`、`ZXW` 等 | 项目自定义点号，应结合表页标题判断 |
| `QDCJ`、`QDWJ`、`QDQXJ` | 某些高架/桥墩模板中的沉降、位移或倾斜点 |

点号含义不清时，优先看表页标题、适配设置页、表头和监测方案。

## 常见监测频率

| 类型 | 常见频率 | 报表写法 |
|---|---|---|
| 自动化监测 | 4 小时/次，常见 6 次/天 | 日报可按当日报告值或日平均值汇总，按项目模板 |
| 人工复核 | 1 次/7 天，风险期可 1 次/3 天或 1 次/天 | 与自动化统计期不同步时要单独说明 |
| 预警/报警期 | 自动化加密至 12 次/天或按指令执行 | 写明实际加密频率和起止时间 |
| 巡视 | 每周或每日 | “未发现新增病害”必须有巡视记录支撑 |
| 控制网复测 | 每月或按方案 | 写明成果沿用或更新 |

## 正负号约定

| 监测项 | 正值 | 负值 |
|---|---|---|
| 道床沉降 | 上抬 | 下沉 |
| 结构沉降 | 上抬 | 下沉 |
| 桥墩/桥面/梁端沉降 | 上抬 | 下沉 |
| 水平位移 | 按项目定义，常见为向基坑或外部施工侧 | 反方向 |
| 收敛 | 按项目模板定义张开或收缩 | 反方向 |
| 桥墩倾斜 | 写明方向或分量 | 反方向 |

报表汇总表下方必须保留正负号说明。

## 默认阈值模型

只有在当前项目没有给出阈值时，才可作为临时参考，并应提示用户确认。

| 监测组 | 预警值 | 报警值 | 控制值 | 速率/数据报警 |
|---|---:|---:|---:|---|
| 地下自动化沉降/位移 | ±5 mm | ±7 mm | ±10 mm | 按方案，可含连续同向趋势或单次速率 |
| 人工道床沉降、上行线水平位移 | ±3 mm | ±4.2 mm | ±6 mm | 常见 ±1.0 mm/d |
| 人工下行线水平位移、远端沉降 | ±5 mm | ±7 mm | ±10 mm | 常见 ±1.0 mm/d |
| 高架/桥梁/桥墩 | 使用当前方案/安评/专家意见 | 使用当前方案/安评/专家意见 | 使用当前方案/安评/专家意见 | 使用当前方案/安评/专家意见 |
| 专家调整值 | 使用批准后的调整值 | 使用批准后的调整值 | 使用批准后的调整值 | 使用批准后的控制变量或速率 |

样例资料中出现的临时调整值只属于对应项目和对应阶段，不能推广。

## 报警字段含义

| 字段 | 含义 |
|---|---|
| `是否综合报警` | 通常指累计值达到报警值或控制值 |
| `是否数据报警` | 速率、连续趋势、突跳或数据质量触发 |
| `预警` | 达到预警值，应通知并关注趋势 |
| `报警` | 达到报警值，应升级处置并复核 |
| `红色预警` / `超控制值` | 达到或超过控制值，通常需要专题研判或专家会 |
