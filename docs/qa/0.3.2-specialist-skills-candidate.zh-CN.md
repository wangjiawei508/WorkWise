日期：2026-07-25（上海时间）

## 范围

本轮验收覆盖：

### 专项回归

- 内置 Skill 安装、修复、来源 metadata 和用户同名非托管包保护。
- Tender Agent 不可修改、最高权限为 `workspace-write`，不允许 Shell 与 MCP。
- 活动会话 Agent 选择、保存中禁用发送、一次精确 revision 冲突重放。
- Tender 货物、软件/服务、工程监测三类样例的要求提取、评分排除、证据和终稿阻断。
- Tender 报价脚本不使用隐式示例数字，不接受负数、无穷值或缺少配置。
- DOCX 只在生成真实 OOXML 时成功，依赖缺失或格式不安全时明确失败。
- Document Illustrator 不包含旧版凭据脚本，不读取第三方 `.env` 或旧 Agent 路径。
- 图片工具把真实 PNG/JPEG/WebP 写到工作区相对路径，并拒绝越界、符号链接和 junction 引用。
- 非视觉模型只能报告文件/格式/尺寸检查，不能声称已经检查图片像素。
- 三个受限来源没有进入 `src/asset/skills`，递归文本扫描也没有发现其固定提交或源码地址。
- 中文和英文市场文案完整；受限项目显示外部链接按钮，不显示误导性的安装按钮。

## 尚未冒充通过的项目

- 当前本机图片生成设置未启用，不能进行真实第三方图片服务的端到端调用。已用确定性测试提供商验证真实图片文件、扩展名规范化、工作区插入路径和失败关闭行为；正式提供商效果仍需在用户配置图片服务后补测。
- 本机 Computer Use 服务在读取 WorkWise 开发版窗口状态时返回 `timeoutReached`，因此本记录不声称实际界面点击冒烟已经通过。
- 三类真实招投标完整项目成果和真实文档配图成果仍保留为候选安装包实际界面验收项。

## 候选准入条件

只有三平台候选安装包确认：

1. 仅包含 Tender Master 和 Document Illustrator 两套新增内置 Skill；
2. 包含 Document Illustrator 的 MIT 许可证与固定来源；
3. 不含 Social Card、Material Illustration、Logo Generator 源码；
4. Tender Agent、市场卡片和中文/英文文案可见；
5. 安装包非空、版本正确、ASAR 内容和 SHA-256 可核验；

并补充实际安装界面证据后，才能报告为可发布候选。不得据此记录直接创建版本标签或公开 Release。
