## 本轮主动发现并修复的问题

1. 打包后的 `better-sqlite3` 曾使用宿主 Node ABI 147，Electron 43 需要 ABI 148。已增加打包前强制重建、安装包冒烟检查和 CI 阻断。
2. Office/PDF 相对路径和 Markdown 文件链接曾被渲染安全层错误拦截。已只放行受控 `deepseek-file:` 协议，并继续拒绝 `file:`、`data:` 和 `javascript:`。
3. XLSX 稀疏表预览曾出现大量 `NaN`、`Unnamed` 和空列。已增加只作用于预览层的稀疏表压缩，解析原始数据不受影响。
4. “请输出相对路径，不要使用 Markdown 链接”曾被误判为文件交付任务，导致多次自动续跑后停滞。已收紧交付意图识别，并用真实安装包验证为单次完成。
