The current WorkWise source tree is based on the last audited MIT commit `363fdf566657cd4d60801f62b0b8f3aa8dfbf2fc`. It does not import the later PolyForm Noncommercial source line. WorkWise features added after that boundary—including the managed CLI center, RailWise knowledge retrieval, presentation workflow, branding, and desktop integration—are maintained as independent WorkWise implementations.

WorkWise has since been renamed, redesigned, and extended for engineering, infrastructure, and operations workflows. DeepSeek-related provider names, model IDs, and API defaults are retained only where they refer to the DeepSeek API rather than the application name.
