WorkWise began from a public MIT-licensed code import of `AdrianAndroid/DeepSeek-GUI`, downloaded from the public `main` branch on 2026-06-13.

The imported project metadata and documentation also referenced `XingYu-Zhong/DeepSeek-GUI` as the original upstream repository. The MIT license notice from the original project is preserved in `LICENSE`.
